Solution 1. (a) Let v=(a,b)\in\mathbb{R}^2. Then \lim_{t\to 0}\frac{f(tv)-f(0)}{t}=\begin{cases}\lim_{t\to 0}\frac{(ta)^2(tb)/[(ta)^2+(tb)^2]}{t} & \text{if } v\neq 0,\\ \lim_{t\to 0}\frac{0}{t} & \text{if } v=0\end{cases} =\begin{cases}\lim_{t\to 0}t\left(\frac{a^2b}{a^2+b^2}\right)=0 & \text{if } v\neq 0,\\ 0 & \text{if } v=0.\end{cases} Thus the limit exists. \square
